# Doublepole

A simple double-pole balancing simulation using [LÃ–VE](https://love2d.org/) and [MoonScript](http://moonscript.org/).


## Instructions

```shell
$ moonc *.moon  # Compile MoonScript
$ love ./       # Run simulation
```

Use left/right arrow keys to move the cart.
Press <kbd>F1</kbd> to reset.
